📊 Controle de Fluxo de Caixa em Excel

Projeto desenvolvido em Excel com o objetivo de estruturar e automatizar o controle de fluxo de caixa, aplicando conceitos de organização financeira, análise de dados e estatística básica.

A planilha foi pensada para uso prático e conta com manual de uso integrado, facilitando a compreensão e aplicação mesmo por usuários não técnicos.

🔹 Funcionalidades

-Registro estruturado de entradas e saídas
-Comparativo orçado x realizado
-Análise de variações financeiras
-Identificação de outliers com apoio estatístico
-Indicadores para acompanhamento e tomada de decisão
-Organização clara das abas e dos dados

📘 Manual de uso

O projeto inclui um manual explicativo, contendo:

- Descrição de cada aba da planilha
- Orientações de preenchimento
- - Explicação dos cálculos e indicadores utilizados
- Boas práticas para uso e adaptação do modelo

🛠️ Tecnologias e conceitos aplicados

- Microsoft Excel / Google Sheets
- Fórmulas avançadas
- Análise estatística básica
- Organização e padronização de dados
- Lógica analítica aplicada a finanças

🎯 Objetivo do projeto

Aplicar, na prática, conhecimentos de Excel e análise financeira, criando uma ferramenta funcional, documentada e reutilizável para controle financeiro pessoal ou administrativo.
